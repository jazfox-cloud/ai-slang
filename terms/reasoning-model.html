What does Reasoning Model mean?

A model marketed or designed to spend extra compute on hard problems before answering, especially math, coding, planning, and multi-step analysis.

A reasoning model is tuned to work longer on problems that need multiple steps. It can be slower or costlier, but useful when the answer needs planning or verification.

Origin and usage

Became mainstream builder vocabulary after model providers started shipping dedicated reasoning models and controls for reasoning effort.

Source type: product-term. Last checked: 2026-07-05.
